## Bulma

Bulma is an open source CSS framework based on Flexbox, and is used by more than 100,000 developers.

Why Bulma?
* 100% Responsive.
* Modular.
* Modern.
* Free (Source code on Github).
* growing community.
* Easy to learn.
* Quick Customization.

To get started, install Bulma using npm as follows (remove the $ "):
```terminal
$ npm install bulma
```
